# 🦖 DinoWeb

Welcome to **DinoWeb**, the official website for the DinoWeb browser game!

DinoWeb lets you play a modern version of the classic dinosaur runner directly in your browser. With fast loading, smooth gameplay, and a clean design, you can jump straight into the action—no downloads or installations required.

* 📝 Release notes and changelog

## 🎮 About DinoWeb

DinoWeb is a lightweight, browser-based endless runner inspired by the classic dinosaur game. Built with modern web technologies, it offers smooth performance, quick loading times, and a fun experience without ads or unnecessary distractions.

## 💻 Technologies Used

* HTML5
* CSS3
* JavaScript
